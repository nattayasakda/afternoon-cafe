# afternoon-cafe

A beautifully designed coffee shop landing page with a complete reservation system.

## Features
- **Modern UI**: Glassmorphism, smooth animations, and pastel color theme.
- **Reservation System**: Book tables with dates, timeslots, and pax selection.
- **Supabase Integration**: Direct insertion into Supabase `reservations` table.
- **Testing**: Jest test suite configured for logic and DOM interactions.

## How to Run
1. Install dependencies: `npm install`
2. Run local server: `npm start`
3. View tests: `npm test`
